# ETL Simple Model

Projeto simples de ETL desenvolvido em Node.js com foco em organização de código, modularização e fluxo de dados.

## Objetivo
Demonstrar um modelo básico de ETL (Extract, Transform, Load), separando responsabilidades e facilitando manutenção.

## Tecnologias
- Node.js
- JavaScript
- Estrutura modular

## Observação
Projeto compartilhado para fins de demonstração de conceito.
